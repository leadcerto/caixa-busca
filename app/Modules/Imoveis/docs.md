# Módulo: Imóveis

Módulo responsável pela gestão, exibição e busca de imóveis.

## Funcionalidades
- Exibição de detalhes do imóvel.
- Filtros de busca avançada.
- Integração com dados de bairro.

## Estrutura
- **Http/Controllers**: Controladores de visualização.
- **Models**: Entidades de Imóvel.
- **Livewire**: Componentes reativos da interface.
